# Harm Reduction Prompt Pack

## Source-backed city guide prompt

> Build a public-safe harm-reduction city guide using only the supplied sources and local official public-health links. Include disclaimers, emergency-resource pointers, and a checklist. Do not include private event locations or unverified claims.

## Correction prompt

> Review this harm-reduction copy for accuracy, stigma, overclaiming, missing sources, and local-law ambiguity. Return suggested edits with citations.

## Care-team onboarding prompt

> Turn this venue/event checklist into a nonmedical volunteer onboarding document. Keep the language nonjudgmental, noncoercive, and approval-gated.

## Fork-demo prompt

> Create a finished demo fork from the framework for a fictional city chapter. Use placeholders, no private locations, no real event claims, and run the verifier.
