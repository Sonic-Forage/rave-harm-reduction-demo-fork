# Lesson: Fork This Into a Local Harm-Reduction Guide

This is the lesson for community builders.

## 1. Fork / copy the repo

Create your own repo from this kit. Name it for your city, crew, campus, or festival lane.

## 2. Pick one safe lane

Start small:

- city resource guide
- safe dance-space checklist
- source-backed correction pack
- DJ/organizer preflight checklist
- volunteer care-team onboarding

## 3. Fill the payload

Use `framework/payloads/harm-reduction-city.payload.example.json`.

## 4. Generate docs, then verify

Use the prompt pack, but require citations and disclaimers. Then run:

```bash
python3 scripts/verify.py
```

## 5. Keep gates closed

Public posting, event-specific outreach, training claims, fundraising, field operations, and medical/legal advice stay closed until a real human organizer approves.

## 6. Open a pull request upstream

If your improvement is general, contribute it back to Sonic-Forage.
