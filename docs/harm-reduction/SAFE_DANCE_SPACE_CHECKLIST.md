# Safe Dance Space Checklist

## Basics

- [ ] Free/visible water access
- [ ] Cool-down/chill area
- [ ] Ear protection encouragement
- [ ] Clear exits and wayfinding
- [ ] Consent and anti-harassment policy
- [ ] Nonjudgmental “ask for help early” messaging
- [ ] Local emergency plan
- [ ] Trained staff/partners identified
- [ ] Public-health resources listed
- [ ] No private scene info in public docs

## Harm-reduction education lane

- [ ] Cite sources
- [ ] Use “education, not medical/legal advice” disclaimer
- [ ] Explain limits of tools such as test strips
- [ ] Keep local legal variation explicit
- [ ] Invite corrections from qualified community partners
