# 🧪 Rave Harm Reduction Demo Fork — Finished Lesson Build

This repo is a **finished demo fork** of the Sonic Forage Harm Reduction Community Kit.

Upstream framework:
https://github.com/Sonic-Forage/rave-harm-reduction-community-kit

This shows what a community builder can do after forking:

1. pick a local/community lane
2. fill a source-backed payload
3. create a safer dance-space guide
4. run verification
5. keep external action closed until human approval

> Demo note: This is a fictional/public-safe example. It does not describe a real event, private location, medical service, or field operation. It is **not medical advice**, legal advice, emergency response training, or permission to use illegal substances.

## Finished demo output

- `demo/EXAMPLE_CITY_CHAPTER.md` — sample chapter output
- `demo/CARE_TEAM_ONE_PAGER.md` — printable-ish care-team summary
- `demo/FORK_REPLICATION_CHECKLIST.md` — repeatable checklist for another public-safe fork lesson
- `demo/LESSON_RECEIPT.md` — what changed from upstream
- `framework/payloads/harm-reduction-city.payload.example.json` — filled with example placeholders

## Run verification

```bash
python3 scripts/verify.py
python3 scripts/verify_demo.py
```

## Lesson

Forking is the culture: the upstream gives the framework, the local crew brings source-backed local context, and the gate stays closed until real humans approve external action.
